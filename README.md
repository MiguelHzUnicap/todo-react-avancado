# todo-react-avancado

Aplicação de lista de tarefas desenvolvida com React utilizando recursos avançados.

## Tecnologias
- React
- JavaScript
- Context API
- Hooks
- Vite

## Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Remover tarefas
- Filtrar tarefas
- Persistência com localStorage

## Como executar
npm install  
npm run dev
